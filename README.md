# 利用方法  
## 商品の購入  
1. トップページ(一覧ページ)から商品の詳細ページへ遷移する。
2. 「カートに入れる」ボタンを押下することで、カートに商品を追加する。
3. カートより、「お支払いへ進む」ボタンを押下すると購入ページへ遷移する。
4. お支払い情報と配送先の情報を入力することで商品が購入できる
